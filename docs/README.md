# Bank System - OOP Project

This project is a Bank Management System implemented using Java.
It applies Object-Oriented Programming concepts and SOLID principles
to simulate real banking operations.

## Project Objectives
- Apply OOP concepts in a real-world system
- Implement SOLID principles
- Use design patterns where applicable
- Build clean, maintainable, and extensible code

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- SOLID Principles
- Strategy Design Pattern
- Builder Pattern

## Main Features
- Multiple account types (Saving, Current, Fixed Deposit)
- Deposit, Withdraw, and Transfer operations
- Interest calculation using Strategy Pattern
- Account status management (Active, Frozen, Closed)
- Flexible account creation using Builder Pattern

## Project Structure
- models: Core business entities (Account and subtypes)
- interfaces: Contracts for operations
- strategy: Interest calculation strategies
- builder: Builders for flexible account creation
- services: Service layer for account operations
- enums: System constants and states
- docs: Project documentation
